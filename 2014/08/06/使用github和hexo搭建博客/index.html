使用github和hexo搭建博客

第一篇文章，记录下这个博客的来历吧。

为什么是Github&Hexo

其实一开始我只是想尝试一下github这个被称作“程序员的Facebook”这个东西的。在了解了github在天才的程序员们手里的各种玩法之后，觉得应该从这个平台上学习些东西，于是希望有个东西能够记录我的成长。正好看到github可以搭建博客，便开始好好利用google嗨起来了。

一开始在网上搜到的最多的教程其实是使用github和jekyll搭建博客，加上官网的推荐，我一开始的尝试也是使用jekyll。可是倒腾了一天，总是有些各种各样的问题。毕竟我是学微电子技术的，CSS神马的完全不知道怎么回事，因此走了不少弯路。以后有机会一定要好好学习下CSS！

后来在知乎上看到了有大神说可以使用github加hexo搭建，于是搜索下相关文章，看起来一副很简单的样子，果断走起！

在此特别感谢 赵伟zippera 博客的帮助！
